## 二、可直接使用的资源

### 1. Botometer 特征体系（强烈推荐）

**来源**: Indiana University Network Science Institute  
**链接**: https://botometer.osome.iu.edu/

Botometer 提供了**超过1000个账户特征**，分为以下类别：

| 特征类别 | 具体指标 |
|---------|---------|
| **时间特征** | 发帖频率、发帖时间规律、活跃时间一致性 |
| **社交网络结构** | 粉丝/关注比、网络连接模式、关系模式 |
| **内容分析** | 推文特征、转发行为、内容模式 |
| **用户资料** | 账户年龄、资料完整度、简介信息 |
| **互动指标** | 互动率、@提及频率、回复模式 |

```python
# Botometer 特征可直接转化为对抗模式
BOTOMETER_PATTERNS = [
    # 时间特征
    "发帖时间呈现高度规律性（如每隔固定分钟发布）",
    "活跃时间与正常用户作息不符",
    "短时间内大量发帖（每分钟超过N条）",
    
    # 社交网络结构
    "粉丝/关注比异常（关注远多于粉丝）",
    "与已知机器人账号有大量关注关系",
    "网络中形成异常紧密的小团体",
    
    # 内容分析
    "内容高度重复或模板化",
    "转发比例异常高（>90%为转发）",
    "内容与账号历史兴趣不符",
    
    # 用户资料
    "账户创建时间与活跃度不匹配",
    "资料信息不完整或使用默认头像",
    "用户名包含大量数字或随机字符",
    
    # 互动指标
    "发帖很多但几乎无人互动",
    "只转发不原创",
    "@提及行为异常（批量@多人）",
]
```

### 2. TwiBot-20 数据集特征

**来源**: 清华大学等  
**链接**: https://github.com/BunsenFeng/TwiBot-20  
**规模**: 229,573用户 + 33,488,192推文

提供三类特征：
- **语义特征**: 推文文本内容
- **属性特征**: 用户元数据
- **邻域特征**: 社交网络结构

```python
# TwiBot-20 提取的行为模式
TWIBOT_PATTERNS = [
    # 语义特征
    "推文内容与特定话题高度相关但缺乏个人观点",
    "使用固定话术模板",
    "文本情感极端（过度正面或负面）",
    
    # 属性特征
    "账号创建日期集中在某一时间段",
    "地理位置信息缺失或不一致",
    "个人简介包含可疑链接",
    
    # 邻域特征
    "与其他可疑账号形成密集互关网络",
    "关注列表高度相似",
    "共同关注异常账号",
]
```

## 四、代码：加载和使用对抗模式库

```python
import torch
from sentence_transformers import SentenceTransformer

class AdversarialPatternLibrary:
    """
    对抗模式库管理器
    """
    def __init__(self, encoder_name='BAAI/bge-large-zh-v1.5'):
        self.encoder = SentenceTransformer(encoder_name)
        self.patterns = self._load_patterns()
        self.pattern_embeddings = self._encode_patterns()
        
    def _load_patterns(self):
        """加载所有对抗模式"""
        all_patterns = []
        
        for category, patterns in COMPREHENSIVE_ADVERSARIAL_PATTERN_LIBRARY.items():
            for p in patterns:
                all_patterns.append({
                    'id': p['id'],
                    'category': category,
                    'name': p['name'],
                    'description': p['description'],
                    'source': p['source'],
                    'text': f"{p['name']}: {p['description']}"
                })
        
        return all_patterns
    
    def _encode_patterns(self):
        """预编码所有模式"""
        texts = [p['text'] for p in self.patterns]
        embeddings = self.encoder.encode(texts, convert_to_tensor=True, normalize_embeddings=True)
        return embeddings
    
    def match_patterns(self, query_text, top_k=5):
        """
        匹配与查询最相关的对抗模式
        """
        query_emb = self.encoder.encode([query_text], convert_to_tensor=True, normalize_embeddings=True)
        similarities = torch.mm(query_emb, self.pattern_embeddings.t())[0]
        
        top_scores, top_indices = torch.topk(similarities, k=top_k)
        
        results = []
        for score, idx in zip(top_scores, top_indices):
            pattern = self.patterns[idx.item()]
            results.append({
                'pattern': pattern,
                'score': score.item()
            })
        
        return results
    
    def get_pattern_embedding(self, pattern_id):
        """获取特定模式的嵌入向量"""
        for i, p in enumerate(self.patterns):
            if p['id'] == pattern_id:
                return self.pattern_embeddings[i]
        return None
    
    def get_category_patterns(self, category):
        """获取某一类别的所有模式"""
        return [p for p in self.patterns if p['category'] == category]


# 使用示例
library = AdversarialPatternLibrary()

query = "找出在社交媒体上通过隐蔽方式协同传播虚假信息的群体"
matched = library.match_patterns(query, top_k=5)

print("匹配到的对抗模式:")
for m in matched:
    print(f"  [{m['pattern']['id']}] {m['pattern']['name']} (相似度: {m['score']:.3f})")
    print(f"      描述: {m['pattern']['description']}")
    print(f"      来源: {m['pattern']['source']}")
    print()
```
